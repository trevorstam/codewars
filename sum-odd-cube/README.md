# CodeWars Challenge: Sum of Odd Cubed Numbers
## Partners: Trevor Stam & Katherine Smith
https://www.codewars.com/kata/sum-of-odd-cubed-numbers

## **Problem Domain**
Find odd numbers in array
Cube odd numbers and sum them.
Function returns undefined if any of the values in the array are NOT numbers

## **Algorithm**
Write function cubeOdd that passes in an array
Declare variable for a new array.
Loop through array
Filter the array only leaving numbers
Check if the length of the new array is NOT equal to the original array
If so, return undefined.
Return the cubed value for each element in the array.
Filter array only leaving the cubed value of the odd numbers
Add the cubed values of odd numbers
